The two projects as complete folders have been shared using Google Drive Links and are to be downloaded as a whole.



1. Run Apache and MySql in XAMPP. Go to Mysql admin

2.Create a new database called 'registration'

3.Run the file sc.sql ( found in folder sql in both folders)  in the top bar 'SQL' after clicking on 'registration' in the left pane.

4.The sc_products table would have been created. Verify if product details are available.

5.Copy the Folder ShopCartNew and ShopCartNew2 to C:// >> xampp>>htdocs 

6. Go to localhost/ShopCartNew or localhost/ShopCartNew2 on the browser.

7. Go to products ... after selection , click on Add to ShopCart on the top right of webpage.
